# AI Agent SOP Library

This folder holds the live, tested SOPs for creating, configuring, fixing, and operating ZedBiz AI agents.

## SOP Groups

- `core/` - Shared standards that apply to all agents and servers.
- `zedbiz-main-vps/` - Main VPS work, including the current Docker-based OpenClaw testing system.
- `zedbiz-secondary-vps/` - Secondary VPS work, including Harry and Edith in the cleaner folder-based format.
- `hermes/` - Hermes agent and orchestration work.
- `shared-scripts/` - Reusable scripts and command snippets used across servers.
- `archives/` - Deprecated or old versions kept for reference.

## Status Labels

- `draft` - Useful, but not fully tested.
- `tested` - Confirmed working in the real environment.
- `needs-fix` - Something failed and needs repair.
- `deprecated` - Kept for history, no longer recommended.
